# How to Add Your Profile Photo

## Quick Steps:

1. **Save your photo** to the `images` folder in this project
2. **Name it exactly**: `profile-photo.jpg` (or `profile-photo.png` if using PNG)
3. **Recommended size**: 600x600 pixels (square format)
4. **File location**: `images/profile-photo.jpg`

## Photo Requirements:

- **Format**: JPG or PNG
- **Size**: Square (1:1 aspect ratio) - 600x600px recommended
- **File size**: Under 500KB for best website performance
- **Quality**: High quality, well-lit, professional appearance

## Image Editing Tips:

If you need to crop/edit your photo:

### Using Online Tools:
- **Photopea** (free, browser-based): https://www.photopea.com/
- **Canva** (free): https://www.canva.com/
- **Remove.bg** (for background removal): https://www.remove.bg/

### Using Windows:
- Use **Paint 3D** or **Photos** app to crop to square
- Right-click image → Edit → Crop to square

### Steps:
1. Open your photo in an image editor
2. Crop to a square (600x600px or larger)
3. Center your face in the frame
4. Save as `profile-photo.jpg` in the `images` folder
5. Make sure the file is optimized (under 500KB)

## After Adding the Photo:

1. The website will automatically display your photo in the About section
2. It will appear as a circular profile picture
3. If the image isn't found, a placeholder will show instead

## Testing:

After adding the photo, open `index.html` in your browser to see if it displays correctly!
