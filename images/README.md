# Images Folder

## Profile Photo

Please add your profile photo here with the filename: `profile-photo.jpg`

### Photo Requirements:
- **Filename**: `profile-photo.jpg` (or `.png`)
- **Recommended size**: 600x600 pixels (square format)
- **Format**: JPG or PNG
- **File size**: Under 500KB for best performance

### How to add your photo:

1. Save your photo in this `images` folder
2. Name it exactly: `profile-photo.jpg` (or `.png` if using PNG format)
3. Make sure the image is cropped to a square format (1:1 aspect ratio)
4. The website will automatically display it in the About section

### Image Optimization Tips:

- Use an image editor to crop it to a square (600x600px recommended)
- Compress the image to reduce file size while maintaining quality
- Ensure good lighting and professional appearance
- The image will be displayed as a circular profile picture

If the image is not found, a placeholder will be shown instead.
